# OpenTaskManager

A Task Manager Software For Linux Systems Inspired By Windows 11 Task Manager

## Requirements

- .NET 10.0 or later
- Linux

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/Junaid433/OpenTaskManager.git/
   cd OpenTaskManager
   dotnet restore
   dotnet build
   sudo dotnet run
   ```
   
- Built with [Avalonia UI](https://avaloniaui.net/)
